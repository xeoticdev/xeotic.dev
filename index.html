About

Xeotic.dev is a private, invite-only project. Access is granted at the owner’s discretion and only after manual review. Nothing on this website is downloadable, and there are no public links to any files.

Reminder: static websites (like GitHub Pages) always expose their own HTML/CSS/JS to visitors. Don’t put sensitive files in this repository.

Access Requirements

  • Join the Discord server.
  • Stay for at least 1 week.
  • Send at least 10 messages while you’re there.

If you break a rule, the waiting time resets to 1.5 weeks from the incident.

Notes

  • Requirements are verified manually. There is no automated verification on this site.
  • There are no downloads here. Any files are shared privately after review.
  • Request access by contacting the owner on Discord.
